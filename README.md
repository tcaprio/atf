# ATF

A Vite + Vue 3 + JavaScript webapp with Vue Router and Vuetify, optimized for mobile use.

## Setup

Install dependencies:
```bash
npm install
```

## Development

Start the development server:
```bash
npm run dev
```

## Build

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Routes

- `/week` - Week view
- `/activities` - Activities view  
- `/history` - History view

The app uses Vuetify for UI components and is designed with mobile-first principles.
